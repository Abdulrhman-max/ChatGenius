                    API Format Requirements
                    

                        Your API must support fetching a single customer by ID:


                        GET {base_url}/{customer_id}


                        Example: if your base URL is https://yoursite.com/api/customers and the customer ID is 42, we call:

                        GET https://yoursite.com/api/customers/42


                        Response must include name, email, and phone fields:

                        {"name": "John Doe", "email": "john@example.com", "phone": "+1234567890"}


                        The API key is sent as both Authorization: Bearer <key> and X-API-Key: <key> headers.


                        Embed usage: Pass the logged-in customer's ID in your embed code:

                        window.ChatGeniusConfig = { ..., customerId: "42" }
